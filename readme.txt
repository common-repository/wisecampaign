== Installation ==

<strong>From Your WordPress Dashboard</strong>

1. Go to Plugins > <strong>Add New</strong>
2. Search for <strong>wiseCampaign</strong>
3. Click on <strong>Install Now</strong> Button
4. Click on <strong>Activate Now</strong> After Installed in the Backend

From WordPress.org

1. Download <strong>[wiseCampaign](https://downloads.wordpress.org/plugin/wisecampaign.zip)</strong>
2. Upload the <strong>'wisecampaign‚Äô</strong> folder to the /wp-content/plugins/ directory
3. Activate <strong> wiseCampaign </strong> plugin from your plugins page


== Usage Instructions ==

1. Go to WiseCampaign -> Settings, toggle the status to Active to activate the banner, and select a template from the list of views. Then, click the Save button.

2. Go to Appearance -> Customize -> wiseCampaign, to customize all the properties of the campaign banner.

## Privacy Policy

Appsero SDK *does not gather any data by default.* The SDK only starts gathering basic telemetry data *when a user allows it via the admin notice*. We collect the data to ensure a great user experience for all our users.
